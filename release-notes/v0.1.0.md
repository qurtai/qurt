## Qurt v0.1.0 — First Release

Your AI coworker, powered by every model. Chat with any supported provider, attach files and images, and bring your own API keys — no subscription, no lock-in.

### Downloads

| Platform | File | Instructions |
|----------|------|--------------|
| **Windows** | `qurt-*.Setup.exe` | Run the installer and follow the setup wizard |
| **macOS** | `qurt-*.dmg` | Open the DMG and drag Qurt to Applications |

### Highlights

- **Multi-model support** — OpenAI, Anthropic, Google, xAI, Moonshot, and more
- **Local-first** — Your API keys and chat history stay on your machine
- **File & image attachments** — Richer conversations with documents and screenshots
- **Agent mode** — Web search, terminal, file editing, and browser control
- **Dark & light themes** — Choose what works for you

### Getting started

1. Download the installer for your platform above
2. Launch Qurt and add your API keys in Settings
3. Pick a model and start chatting

For more details, see the [README](https://github.com/eabdullin/qurt#readme).
